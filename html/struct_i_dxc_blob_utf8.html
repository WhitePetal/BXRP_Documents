Detailed Description

A blob containing a UTF-8 encoded string.

The value returned by GetBufferSize() is the size of the buffer, in bytes, including the null-terminator.

Depending on the -encoding option passed to the compiler, this interface is used to return string output blobs, such as errors/warnings, preprocessed HLSL, or other text. Output name strings always use IDxcBlobWide.
