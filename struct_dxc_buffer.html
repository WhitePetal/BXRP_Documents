Detailed Description

Interface for handling include directives.

This interface can be implemented to customize handling of include directives.

Use IDxcUtils::CreateDefaultIncludeHandler to create a default implementation that reads include files from the filesystem.

Structure for supplying bytes or text input to Dxc APIs.
