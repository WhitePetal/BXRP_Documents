5// Example volume switching script
6//
7// The high-level logic is following:
8//
9// - Volumes overlap each other a bit, so we don't need blending! The size of the overlap is the size of your largest dynamic object.
10// - As object enters the volume, set volume data to it. Increment the counter.
11// - As object leaves the volume, decrement the counter. If it equals 0, use global volume (set empty property block).
12// - If the volume is moving, set volume data every frame, in LateUpdate.
13//
