Detailed Description

A blob containing a null-terminated wide string.

This uses the native wide character encoding (utf16 on Windows, utf32 on Linux).

The value returned by GetBufferSize() is the size of the buffer, in bytes, including the null-terminator.

This interface is used to return output name strings DXC. Other string output blobs, such as errors/warnings, preprocessed HLSL, or other text are returned using encodings based on the -encoding option passed to the compiler.
